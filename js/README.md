1.两数之和
[题目]：给出一个数组和一个目标值，在该数组中找出和为目标值的两个数，并且这两个数在数组中的下标不同。
[示例]:
输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1]
